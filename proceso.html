Proceso 📜🌱


1. Recolección

El reciclaje de papel comienza con la recolección del papel usado, que puede ser periódico, cartón o papel de oficina.

Este material se recolecta a través de contenedores específicos ubicados en diferentes lugares, como hogares y empresas.

La recolección adecuada es fundamental para garantizar que el material reciclable llegue a las plantas de reciclaje.


2. Clasificación y Limpieza

Una vez recolectado, se transporta a una planta de reciclaje donde se clasifica por tipo y calidad.

Durante este proceso, se eliminan contaminantes como grapas, plásticos y otros materiales no reciclables.

La limpieza del papel es crucial para asegurar que el producto reciclado final sea de alta calidad.


3. Trituración y Mezcla

Después de la limpieza, el papel se tritura y se mezcla con agua, formando una pulpa que facilita el proceso de reciclaje.

Se añaden productos químicos para descomponer los tintes y otros aditivos presentes en el papel original.

Este paso es esencial para preparar la pulpa para su posterior filtrado y limpieza.


4. Filtrado

La pulpa se filtra para eliminar impurezas y obtener una mezcla homogénea lista para ser procesada.

Este proceso asegura que el papel reciclado sea libre de contaminantes y tenga una consistencia uniforme.

Después del filtrado, se procede a la formación de hojas de papel.


5. Secado y Corte

La pasta de papel se extiende en hojas finas y se seca, creando nuevas hojas de papel reciclado.

Este proceso incluye la compresión para eliminar el exceso de agua y asegurar un secado adecuado.

Finalmente, el papel reciclado se corta y se empaqueta para su distribución en el mercado.


6. Beneficios

Este proceso no solo ahorra recursos naturales, sino que también contribuye a la sostenibilidad ambiental.

Al reciclar papel, se reduce la necesidad de talar árboles y se disminuye la generación de residuos.

Promover el reciclaje es clave para un uso responsable de los recursos en nuestra sociedad.
